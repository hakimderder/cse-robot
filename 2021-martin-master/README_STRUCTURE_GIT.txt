Structure du depot:
===================

- admin      -- Information administrative, cahier des charges, ..
  |
  |- meeting      PV des réunions hebdomadaire (journal de travail)	
  |
  |- planning     plannings du travail de bachelor
  |
  |- ...

-doc         -- Documentation de tiers, merci de faire un classement (sous-répertoires)
  |
  |- ros          docs sur ROS, ...
  |
  |- datasheet    datasheet des composants électroniques
  |
  |- ...


-dev         -- Developpement realise lors du TB, strucutre à définir 
  |                  
  |- ...

-publi       -- Documentation redigée lors du TB (document que vous rédigez)
  |
  |- images             images utilisées pour le rapport et la documentation
  |
  |- presentations      présentation pour la défense
  |
  |- rapports           rapport final du projet, rapport-intermédiaire du projet
  |
  |- tutorials          tutoriels réalisés
  |
  |- ...
